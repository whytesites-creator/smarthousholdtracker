# Smart Household Tracker: Family Module Usage Guide (With Sample Data)

This guide shows how a real family can use every module in day-to-day life.

## 1) Sample Family Profile

Use this sample while onboarding:

- Household name: `Bala Family - Chennai`
- Owner: `Bala`
- Members:
  - `Bala` (self)
  - `Priya` (spouse)
  - `Anika` (child)
  - `Raman` (parent)

## 2) First-Day Setup Flow

1. Register and login.
2. Create household: `Bala Family - Chennai`.
3. Invite spouse/adult members by email.
4. Open each module and add at least one record.
5. Open `Notifications` and click `Refresh` to generate alerts.

## Module K: Notifications

### What to track
Auto alerts from bills/vehicle/appliance/health/documents.

### How to use
1. Open `Notifications`.
2. Click `Refresh` to generate current alerts.
3. Mark items as read after action.

### Example generated alerts
- `Bill Overdue: Electricity`
- `PUC Expiring: Family Car`
- `Warranty Expiring: Kitchen Fridge`
- `Health Reminder: BP Tablet - Raman`
- `Document Expiring: Car Insurance Policy`

## Module L: Dashboard

### What to monitor weekly
- This month expense total
- Low stock count
- Bills due/overdue
- Health pending reminders
- Vehicle expiry warning banner (if any)

### Weekly family review (10 minutes)
- Sunday evening: owner/admin reviews dashboard.
- Assign actions for upcoming week:
  - pay bills
  - refill stocks
  - service booking
  - doctor visit

## 4) Suggested Monthly Operating Rhythm

- Day 1-5: update bills and expected expenses.
- Weekly: inventory + water/gas updates.
- Daily: add expenses and complete health reminders.
- Month-end: review dashboard trends and export notes manually.

## 5) Quick Starter Dataset (Copy Into App)

Use this as your first 30-minute setup checklist:

- Create 4 members.
- Add 5 expenses.
- Add 4 inventory items.
- Add 2 gas/water entries each.
- Add 4 recurring bills.
- Add 1 vehicle + 1 service log.
- Add 3 appliances + 1 service log.
- Add 3 health reminders.
- Add 4 documents with at least 2 expiry dates.
- Open notifications and click `Refresh`.

When this is done, all dashboard cards and alert flows will start showing real data.
